# SourcingOS Career Match V1

Candidate-facing recruiter career matcher for `/career-match`.

V1 is intentionally trust-first:

- deterministic resume parsing
- no fake PDF/DOCX extraction
- no auto-apply
- no invented resume achievements
- no verified-clearance claims
- matches against the existing `/api/jobs/search` job surface

This README exists to document the feature foundation and trigger a clean preview deployment after the additive files landed on the `career-match-v1` branch.
